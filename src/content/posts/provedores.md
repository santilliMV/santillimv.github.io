# Modelado de Sistema de Proveedores y Pedidos

Este es un modelado entidad-relación derivado de un esquema de base de datos para la gestión de proveedores, clientes, pedidos y elementos.

## Componentes Principales

- Proveedores
- Clientes
- Pedidos
- Elementos

## Características

El modelo permite gestionar la relación entre proveedores que ofrecen artículos a distintos precios, clientes que realizan pedidos con cantidades específicas, y el catálogo de elementos con sus descripciones.
