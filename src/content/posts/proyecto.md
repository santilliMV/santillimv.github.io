# Modelo Entidad-Relación – Sistema de Gimnasio

Diagrama MER diseñado para gestionar las operaciones de un gimnasio.

## Descripción

Modelo de base de datos que cubre desde la gestión de miembros y membresías hasta el control de clases, salas, equipos y empleados.

## Entidades principales

- Miembros, acudientes y membresías por plan
- Clases, reservas y salas
- Equipos y mantenimientos
- Empleados (instructor, técnico, recepcionista, administrador)
- Pagos generados por membresía
