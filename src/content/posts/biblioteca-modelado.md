# Modelado de Biblioteca

Modelo entidad-relación para un sistema completo de gestión de biblioteca.

## Entidades

- Libros
- Autores
- Usuarios
- Préstamos
- Categorías

## Funcionalidades

Sistema que permite registrar, gestionar y prestar libros de la biblioteca.
