# Modelado de Biblioteca Comunitaria

Este es un modelado entidad-relación para un sistema de gestión de una biblioteca comunitaria local que administra préstamos de distintos tipos de artículos a sus usuarios.

## Componentes Principales

- Artículos (Libros, CD, Cintas, etc.)
- Usuarios
- Préstamos
- Patrocinadores

## Características

El modelo permite gestionar el inventario de artículos disponibles, los usuarios con sus datos personales, los préstamos con fechas límite, las penalizaciones por devoluciones tardías y la relación de tutoría entre patrocinadores y usuarios menores de edad.
