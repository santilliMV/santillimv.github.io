# Modelado de Oficina de Registro Universitario

Este es un modelado entidad-relación para el sistema de registro académico de una universidad que gestiona asignaturas, ofertas académicas, estudiantes, profesores y matrículas.

## Componentes Principales

- Asignaturas
- Ofertas de Asignatura
- Estudiantes
- Profesores
- Matrículas

## Características

El modelo permite registrar el catálogo de asignaturas con sus prerrequisitos, las ofertas por semestre con horarios y aulas, la información de estudiantes y profesores por departamento, y las matrículas con las notas obtenidas por cada estudiante en cada asignatura cursada.
